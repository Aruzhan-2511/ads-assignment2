## Short Summary

This assignment implements a mini banking system using basic data structures.

- A LinkedList is used to store bank accounts and perform operations like adding accounts, searching by username, depositing, and withdrawing money.
- A Stack (LIFO) is used to manage transaction history, allowing users to view and undo the last transaction.
- A Queue (FIFO) is used for bill payments, processing requests in order.
- Another Queue is used to simulate account opening requests, where users submit requests and the admin processes them.
- An array is used to store a fixed number of predefined bank accounts.
- A menu-driven system combines all features, including Bank operations, ATM services, and Admin controls.

Overall, the project demonstrates practical usage of LinkedList, Stack, Queue, and Array in a real-life banking scenario.
